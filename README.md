# Styled components example

Button example with styled components with
webpack, react and styled-components.

## Run

With `yarn` or `npm` run:

```sh
yarn run start-dev
npm run start-dev
```

## Test

With `yarn` or `npm` run:

```sh
yarn test
npm test
```
